# Инструкция по получению GigaChat API

Этот документ описывает шаги для регистрации на платформе SberDevelopers и получения API-ключа для GigaChat.

## Шаг 1: Регистрация аккаунта

- **Действие**: Зарегистрируйте аккаунты для тьютора и ребенка.
- **Ссылка**: [Перейти на страницу регистрации](https://developers.sber.ru/studio/registration)

![Процесс регистрации](./img/1.png)

## Шаг 2: Получение API-ключа

- **Действие**: После регистрации создайте проект в личном кабинете, чтобы получить ключ GigaChat API.
- Это необходимо для аутентификации ваших запросов к сервису.

![Создание проекта](./img/2.png)

![Получение ключа](./img/3.png)

## Важная информация

- **Тарифы**: Ознакомьтесь с актуальными тарифами для индивидуальных пользователей по [этой ссылке](https://developers.sber.ru/docs/ru/gigachat/tariffs/individual-tariffs).
